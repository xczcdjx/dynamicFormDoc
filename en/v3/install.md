# dynamicformdjx

## Install

```bash
# Choose any of the following
npm install dynamicformdjx
# or
yarn add dynamicformdjx
# or
pnpm add dynamicformdjx
```

## Component Notes

1. The current version has completed full adaptation for forms used with Element Plus or Naive UI.
2. Simple forms, custom forms, and decorated forms are all supported.
3. The documentation is still being improved. For specific usage examples, please refer to the `components` directory in the source code.
